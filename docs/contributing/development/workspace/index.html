Workspace

Cepi is divided into the server jar, Sabre (using Minestom and all of its extensions.

To install the server + all its extensions, use:

Shell (Mac, Linux)

curl -fsSL https://installsh.cepi.world | sh

Powershell (Windows)

iwr "https://installps.cepi.world" | iex
